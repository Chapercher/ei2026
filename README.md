# Entretien Individuel 2026 — Quentin Charton

Site de rétrospective gamifiée pour mon entretien individuel 2026, fait avec Claude IA.

## Stack
- [Astro](https://astro.build)
- Docker (dev local)
- GitHub Pages (déploiement)

## Lancer en local

```bash
docker compose up -d
```

Puis ouvrir <http://localhost:4321/>

## Déploiement

Push sur `main` → déploiement automatique via GitHub Actions sur :
<https://chapercher.github.io/ei2026/>
